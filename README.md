# Shinobu

Work in progress

Shinobu is a Japanese reader app for learners, which automatically annotates plain Japanese text with furigana based on the selected Japanese level, and allows for quickly looking up wortds, translating sentences, and getting explanations.

Supports granular customization of reading experience, such as horizontal vs vertical text, line spacing, font, etc. and automatically paginates text for a book-like experience.

Dependencies:
 - Kawazu
